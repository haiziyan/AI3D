# 快速入门指南

欢迎使用 AI 3D CAD Studio！本指南将帮助你在 5 分钟内开始使用。

## 📋 前置要求

- Node.js 12+ 或任何 HTTP 服务器
- 现代浏览器（Chrome、Firefox、Edge、Safari）
- Supabase 账户（免费）
- DeepSeek API 密钥（按量付费）

## 🚀 5 分钟快速开始

### 步骤 1: 克隆或下载项目

```bash
git clone https://github.com/yourusername/AI3D.git
cd AI3D
```

### 步骤 2: 安装依赖

```bash
npm install
```

### 步骤 3: 配置服务

#### 3.1 创建 Supabase 项目

1. 访问 https://supabase.com/
2. 点击 "Start your project"
3. 创建新项目（选择免费套餐）
4. 等待项目初始化完成

#### 3.2 设置数据库

在 Supabase 控制台：
1. 点击左侧 "SQL Editor"
2. 点击 "New query"
3. 复制 `SETUP.md` 中的 SQL 代码
4. 点击 "Run" 执行

#### 3.3 获取 API 密钥

在 Supabase 控制台：
1. 点击左侧 "Settings" → "API"
2. 复制 "Project URL"
3. 复制 "anon public" key

#### 3.4 获取 DeepSeek API Key

1. 访问 https://platform.deepseek.com/
2. 注册并登录
3. 点击 "API Keys"
4. 创建新密钥并复制

#### 3.5 配置应用

```bash
# 复制配置模板
cp js/config.example.js js/config.js

# 编辑配置文件
# Windows: notepad js/config.js
# Mac/Linux: nano js/config.js
```

填入你的配置：
```javascript
const CONFIG = {
    SUPABASE_URL: 'https://xxxxx.supabase.co',
    SUPABASE_ANON_KEY: 'eyJhbGc...',
    DEEPSEEK_API_KEY: 'sk-...',
    // 其他配置保持默认
};
```

### 步骤 4: 启动应用

#### Windows
```bash
start.bat
```

#### Mac/Linux
```bash
chmod +x start.sh
./start.sh
```

#### 或使用 npm
```bash
npm start
```

浏览器会自动打开 http://localhost:8080

### 步骤 5: 开始使用

#### 5.1 注册账户

1. 点击右上角 "注册"
2. 输入邮箱和密码
3. 查收验证邮件（可能在垃圾邮件中）
4. 点击邮件中的验证链接

#### 5.2 登录

1. 点击右上角 "登录"
2. 输入邮箱和密码
3. 登录成功后会显示积分余额（100）

#### 5.3 使用 AI 生成

1. 在顶部输入框输入描述，例如：
   ```
   创建一个边长50的立方体，中间挖一个半径20的球形孔
   ```

2. 点击 "生成" 按钮

3. 等待 AI 生成代码（通常 5-10 秒）

4. 自动切换到 "3D 视图" 查看结果

5. 如需修改，切换到 "代码编辑器" 手动调整

## 💡 使用技巧

### 1. 高效使用 AI

- **从简单开始**: 先生成基础形状，再逐步添加细节
- **提供尺寸**: 明确指定尺寸，避免模糊描述
- **分步描述**: 复杂模型分多个步骤描述
- **参考示例**: 查看 `AI_PROMPTS.md` 获取更多示例

### 2. 代码编辑

- **自动补全**: 输入函数名时会显示提示
- **快捷键**: 
  - F5: 评估代码
  - Ctrl+S: 保存项目
  - Ctrl+F: 查找
  - Ctrl+H: 替换

### 3. 3D 视图操作

- **旋转**: 鼠标左键拖动
- **平移**: 鼠标右键拖动
- **缩放**: 滚轮或双指缩放
- **重置视图**: 刷新页面

### 4. 导出模型

- **STEP**: 用于 CAD 软件（推荐）
- **STL**: 用于 3D 打印
- **OBJ**: 用于 3D 建模软件

### 5. 节省积分

- **缓存代码**: 保存常用的代码片段
- **手动调整**: 生成后手动微调，避免重复生成
- **批量生成**: 一次描述多个相似的形状

## 🔧 常见问题

### Q: AI 生成失败怎么办？

A: 
1. 检查网络连接
2. 确认 API 密钥正确
3. 简化描述，分步生成
4. 查看控制台错误信息

### Q: 积分用完了怎么办？

A: 
1. 点击用户中心的"充值积分"（开发中）
2. 或联系管理员手动添加

### Q: 如何保存项目？

A: 
1. 点击顶部菜单 "保存项目"
2. 选择保存位置
3. 文件会保存为 .json 格式

### Q: 如何加载项目？

A: 
1. 点击顶部菜单 "加载项目"
2. 选择 .json 项目文件
3. 项目会自动加载

### Q: 移动端可以使用吗？

A: 
可以！界面已完全适配移动端，但 3D 渲染性能可能受限。

### Q: 支持离线使用吗？

A: 
基本功能支持离线（PWA），但 AI 生成需要网络连接。

## 🎓 示例项目

### 项目 1: 手机支架

```javascript
// 底座
let base = Box(100, 80, 10);

// 支撑臂
let arm = Box(10, 80, 80);
arm = Rotate([1, 0, 0], -15, arm);
arm = Translate([45, 0, 10], arm);

// 手机槽
let slot = Box(70, 15, 90);
slot = Translate([10, 32.5, 15], slot);

// 组合
let stand = Difference(Union([base, arm]), slot);
stand;
```

### 项目 2: 简单齿轮

```javascript
let teeth = Slider("齿数", 16, 8, 32);
let outerR = Slider("外径", 40, 20, 60);
let innerR = Slider("内径", 15, 5, 30);

// 外圆
let outer = Cylinder(outerR, 10, true);

// 内孔
let inner = Cylinder(innerR, 12, true);

// 齿（简化版）
let toothAngle = 360 / teeth;
let toothWidth = 2 * Math.PI * outerR / teeth * 0.3;

let gear = Difference(outer, inner);
gear;
```
