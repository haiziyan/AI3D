## 📋 提示词编写技巧

### ✅ 好的提示词特征

1. **明确具体** - 提供准确的尺寸和参数
   - ✅ "创建一个边长50的立方体"
   - ❌ "创建一个立方体"

2. **结构清晰** - 按逻辑顺序描述
   - ✅ "创建一个底座100x100x20，上面放一个半径30的球体"
   - ❌ "球体和底座，球体半径30，底座100x100x20"

3. **使用专业术语** - 使用 CAD 相关术语
   - ✅ "创建一个圆柱体，底面半径20，高度100"
   - ❌ "创建一个圆的长条"

4. **分步描述** - 复杂模型分步骤描述
   - ✅ "首先创建一个立方体，然后在中心挖一个圆柱形孔，最后给所有边添加圆角"
   - ❌ "创建一个带孔和圆角的立方体"

### ❌ 避免的问题

1. **过于模糊** - "创建一个好看的模型"
2. **缺少尺寸** - "创建一个大立方体"
3. **过于复杂** - 一次性描述太多细节
4. **不切实际** - "创建一辆完整的汽车"（太复杂）

## 🎓 进阶示例

### 参数化设计

```
创建一个参数化的螺栓：
- 头部：六角形，对边距离用滑块控制(10-30)
- 杆部：圆柱形，直径8，长度用滑块控制(20-100)
- 螺纹：简化为圆柱形
```

### 组合设计

```
创建一个简单的机器人：
- 身体：立方体 40x40x60
- 头部：球体，半径25，位于身体顶部
- 手臂：两个圆柱体，直径10，长度50，从身体两侧伸出
- 腿部：两个圆柱体，直径12，长度60，从身体底部向下
```

### 实用零件

```
创建一个手机支架：
- 底座：长方体 100x80x10
- 支撑臂：从底座后部向上倾斜45度，高度80
- 手机槽：在支撑臂顶部，宽度70，深度10
```

## 💬 获取帮助

如果 AI 生成的结果不理想：

1. **细化描述** - 添加更多细节和尺寸
2. **分步生成** - 先生成基础形状，再逐步添加细节
3. **参考示例** - 使用本文档中的示例作为模板
4. **手动调整** - 在生成的代码基础上手动修改
