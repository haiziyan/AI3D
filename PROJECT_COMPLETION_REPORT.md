## 🎯 核心功能

### 1. AI 代码生成

```
用户输入: "创建一个带孔的立方体"
↓
DeepSeek API 处理
↓
生成可执行代码
↓
自动渲染 3D 模型
↓
扣除相应积分
```

### 2. 用户系统

```

↓

```

## 🛠️ 技术实现

### 前端技术栈

- **3D 渲染**: Three.js
- **CAD 内核**: OpenCascade.js
- **代码编辑**: Monaco Editor
- **布局系统**: Golden Layout
- **UI 控件**: Tweakpane

### 后端服务

- **认证**: Supabase Auth
- **数据库**: Supabase PostgreSQL
- **AI 服务**: DeepSeek API

### 开发工具

- **包管理**: npm
- **服务器**: http-server
- **版本控制**: Git
